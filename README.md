# 專案: Asioyo 
  - 提供匯率轉換API

## 專案初始化
  - 確認 `node` 版本為 `package.json` 裡面的 `engines` 欄位的指定版本: `node --version`
    - 若版本不同請切換 `node` 版本
      - `nvm use` (版本由檔案 `.nvmrc` 設定)
  - 安裝套件 `npm install`
  - 本機啟動專案：
    - 安裝 `supervisor` 套件到全域: `npm install supervisor -g`
    - 啟動專案: `supervisor server.js`
    - 執行測試碼: `npm run test`
